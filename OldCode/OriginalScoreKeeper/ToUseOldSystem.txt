Copying data from New to Old Format
-----------------------------------

This is written In case of difficulties using the new format of the Scorekeeper
Access DB in 2004

- * Extract (unzip) the 2003 system directory into a folder
- * Copy the extracted Empty.md$ (old format) and rename appropriately
- Using Microsoft Access copy all rows from the new formatted DB tables to the old
  created above (Note - DO ALL tables including CompDtls and Predefined Scores)

     - done by:
     
          - Select ALL records in New DB Table
          - right click while mouse is in a table cell and hit "Copy"
          - open New DB table, right click in the far left hand column
            and select "Paste"

- Once done, excute the old Scorekeeper.exe extracted in step 1
